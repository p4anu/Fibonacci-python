INFORMATION ABOUT FIBONACCI SEQUENCE PY
It is a basic algorithm containing 3 operations for the user: 1) Read information about the Fibonacci sequence. 2) Investigate whether or not an entered number belongs to the Fibonacci sequence. 3) Recreate the original Fibonacci sequence, or create a fake sequence but based on the original formula (with the source numbers entered by the user). 
This work was done by Miguel Ángel De Zan (p4anu)

INFORMACIÓN SOBRE FIBONACCI SEQUENCE PY
Es un algoritmo básico que contienes 3 operaciones para el usuario: 1) Leer información sobre la secuencia de Fibonacci. 2) Investigar si un número ingresado pertenece, o no, a la secuencia de Fibonacci. 3) Recrear la secuencia Fibonacci original, o crear una falsa secuencia pero que esté basada en la fórmula original (con los números de origen que ingrese el usuario). 
Este trabajo fue hecho por Miguel Ángel De Zan (p4anu)
